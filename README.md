B. OVERVIEW
A responsive cryptocurrency dashboard built with HTML, CSS, and JavaScript, featuring a modern UI for tracking digital assets. This project focuses on clean design and interactive elements, ensuring a fast and lightweight user experience across all devices.

C. FEATURES
- Real-time UI for monitoring cryptocurrency prices
- Interactive data representation and smooth transitions
- Fully responsive layout for mobile and desktop browsers
- Clean and modern crypto-themed aesthetic

D. TECH STACK
- Frontend: HTML5, CSS3
- Scripting: JavaScript (ES6+)
- Design: Responsive Web Design

E. RULES
- Open index.html in any web browser to launch
- Navigate through the dashboard to view asset data
- Use the sidebar/menu to switch between different market views
